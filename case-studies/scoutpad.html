Case Study

ScoutPad – Turning a Wordpress Theme into a Custom-Designed Landing Page

ScoutPad is a new tool by Matchmetrics that enables professional football scouts to create match reports easily and quickly. I’ve been hired by Matchmetrics to redesign and develop their marketing website. Due to the upcoming product launch, the main challenge of this project was to turn a WordPress theme into a custom design fitting to Matchmetrics brand in a short amount of time.

The Challenge

Customzing a WordPress Theme

When I was hired by Matchmetrics for this project, they already had a pre-launch website for ScoutPad. This website was running on WordPress using the Uplift theme. Due to the upcoming launch of ScoutPad and the limited budget of this project, we decided to build the new website based on this theme.

While this decision limited the possibilities to create a completely custom design, it helped to save crucial time during the development phase. Furthermore, it also helped to save time for the client, since they were already familiar with the content management system provided by the theme.

Design Process (2/2)

Customizing the Modules

After selecting the design direction, I started to explore the available modules of the theme and matching these to the provided content.

Once all the modules were selected, I started to customize their design to fit Matchmetrics look and feel. In this process, it came in very handy to be the designer and developer of the project, as I occurred able to have a look at the existing code to evaluate the development cost of certain design changes.
