The problem

A while ago, dribbble changed their design. Instead of dividing their feed into multiple pages, they started to use infinite scrolling. While this definitely has several advantages, it can cause hiccups when navigating back and forth in your browser. By default, user profiles and links in shot descriptions are opened in the same tab - if you open one of them and afterward navigate back to the feed, you often don't end up at the previous scroll position.

The Solution

It's a small detail, but I consistently annoyed me. Therefore I decided to develop tabbbs, a tiny chrome extension that ensures those links are always opened in a new tab.


You can download it for free at the Chrome Web Store.

How it works

The extension works by simply adding a target="_blank" to the respective links via JavaScript. The whole project is open source, you can find the code on GitHub.


By default, each link to a user profiles and each link in a shot description is opened in a new tab. However, you can change that in the settings.
