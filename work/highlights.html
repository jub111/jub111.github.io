Highlights

App Concept

Imagine you visit a friend who lives in a town you have never been before. Sure your friend will tell you about all the popular sights, but furthermore he will guide you to all his beloved spots: The most delicous burger in town, the neat little cafe just around the corner, the new underground club and the best park to hang out. Highlights is not about the popular sights, it’s all about those secret gems that make a town unique.

User Stories

We wanted to create a minimalistic app which has yet all the features necessary. User stories helped us to abstract all the requirements into small chunks and discuss their individual necessity and benefits as well as their relationsship to other user stories.

User wants to search for places nearby while visiting a new town

User wants to search for places in a specific town while planing a trip

User wants to filter by categories while searching for places

User wants to save a place for later while browsing places

User wants to share a place with friends while browsing places

User want to create a new place while or after beeing at this place

User want to add a photo to a place while or after beeing there

User want to edit a place while or after beeing there

Exploring New Patterns

Since this is a side project with no limitations by budget or time, we challenged ourselves to create something new. Our goal was to create an app with excellent user experience that feels unique the second you open it. In order to get there we left all the commonly used patterns behind us and started exploring. Based on our sketches we build some rough prototypes.

The Current Design

By iterating through many sketches and prototypes we finally found a design that met our requirements. This project is still work in progress, right now we're working on some further screens and the interactions. In the following our current progress is shown.
